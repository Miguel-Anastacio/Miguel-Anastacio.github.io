Grand Strategy Style Map Plugin

A plugin for Unreal Engine published on the Epic Games Marketplace. It streamlines the development of Grand Strategy style maps, like the ones present in games such Europa Universalis IV or Hearts of Iron IV, by providing the behaviour necessary to interact and navigate the map.

Main features include:

• Map generation based on user provided data.

• Interactive map split into provinces.

• Map navigation controls in spherical and flat maps

Project Few

Dogfighting game with semi-realistic flight physics. The gameplay prioritizes fun while still retaining a significant degree of scientific accuracy. Features a AI capable of adapting to the terrain and engage the player, its behaviour is controlled by a state machine. The game loop is a team deathmatch style round, with 29 AI controlled planes and a player.

Self Driving Cars

A genetic algorithm that teaches cars how to steer and brake to avoid hitting walls, as well as complete a track in the fastest time. The genetic algorithm simulates a population of fifty cars that each have different genes which affect values like the car's ability to turn away from walls, vision and speed.

Football Manager Player Analyzer

An application that allows users to compare and analyze Football Manager(FM) players and their efficiency at all the roles of the game. Users can upload multiple players data through files. It uses GLFW for the window and ImGui for the UI.
